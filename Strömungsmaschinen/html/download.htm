Windows Media Player  
Windows Media Player ist eine Abspielsoftware von Microsoft für Audio- und Videodateien. Dieser Medienplayer ist für Microsoft Windows und Mac OS verfügbar.

VLC Media Player  
Der VLC Media Player zählt zu den beliebtesten Wiedergabe-Programmen für Multimedia-Inhalte. Sehr gut geeignet für die Wiedergabe von vielen verschiedenen Video- und Audioformaten. Es müssen keine weiteren Codecs installiert werden, da der VLC-Client bereits alle Codecs integriert hat.

QuickTime Player  
Mit QuickTime für Windows von Apple lassen sich u. a. QuickTime-Filme mit der Dateiendung MOV abspielen. QuickTime enthält ein Plug-In für den Internet Explorer sowie Netscape. Damit lassen sich QuickTime-Filme, die in eine Webseite integriert sind, direkt im Browser abspielen.

Adobe Shockwave Player  
Mit dem Shockwave Player für Windows von Adobe lassen sich Animationen mit der Dateiendung SWF abspielen.

RealPlayer  
Die kostenlose Grundversion des RealPlayers kann u. a. die Dateiformate MPG und MOV abgespielen.
PowerPoint Viewer  
Microsoft Office PowerPoint Viewer 2007 ist ein Programm, das zum Ausführen von Präsentationen auf Computern verwendet wird, auf denen Microsoft PowerPoint nicht installiert ist.
